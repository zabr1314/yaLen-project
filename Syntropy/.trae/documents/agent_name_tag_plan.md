# Agent 身份铭牌显示计划 (Agent Name Tag)

陛下，既然六部尚书现在穿着相同的官服，为了让您和用户能一目了然地分辨出谁是谁，微臣建议在每位官员的**脚下**添加一个精美的**身份铭牌**。

## 1. 设计思路
*   **位置**：显示在角色 Sprite 的正下方（脚底）。
*   **样式**：
    *   **背景**：半透明的深色圆角矩形或卷轴样式，既清晰又不喧宾夺主。
    *   **文字**：显示官员的中文职称（如“兵部”、“丞相”、“史官”）。
    *   **颜色**：不同部门可以使用不同的文字颜色或边框颜色进行区分（参考 `COURT_ROLES` 中的颜色定义）。
*   **跟随**：铭牌必须跟随角色移动，就像影子一样。

## 2. 实施步骤

### 2.1 修改 `Agent` 类 (`src/game/Agent.ts`)
*   **新增属性**：`nameTag: Phaser.GameObjects.Container`。
*   **初始化**：在 `constructor` 中创建 Container，包含：
    *   背景 `Graphics` (圆角矩形)。
    *   文字 `Text` (从 `COURT_ROLES` 获取中文职称)。
*   **层级管理**：铭牌应该位于角色图层之上（遮挡脚部）还是之下（踩在脚下）？建议位于**角色之上**，避免被背景杂物遮挡，且使用 `setDepth` 确保其随角色 Y 轴排序。
*   **同步移动**：在 `preUpdate` 或 `setPosition` 中，同步更新 `nameTag` 的位置。

### 2.2 职称映射 (`src/constants/court.ts`)
*   确保 `COURT_ROLES` 中已经定义了每个 `AgentId` 对应的中文 `title`。
    *   `minister` -> "丞相"
    *   `official_war` -> "兵部"
    *   `historian` -> "史官"
    *   ...

### 2.3 视觉微调
*   **字体**：使用与 UI 一致的字体（如 serif），字号不宜过大（12-14px）。
*   **偏移量**：根据不同角色的缩放比例（0.35x vs 2.8x），调整铭牌在 Y 轴上的偏移量，确保它刚好贴在脚下。
