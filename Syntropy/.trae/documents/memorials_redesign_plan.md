# 奏折阁 (Memorials Archive) 视觉与体验优化方案

用户反馈“奏折阁”中存在英文、难以理解的 ID 编号（如 #6810），且整体色调过白。本方案旨在进行全面的本地化处理、编号美化，并将其重构为更具质感的深色调古风界面。

## 核心设计思路

1.  **完全本地化 (Full Localization)**: 将所有状态、页脚、空状态提示等从英文改为正式的公文术语。
2.  **编号雅致化 (Elegant ID)**: 将原始 ID（如 #6810）重构为“第 XXXX 号”或“奉天承运第 XXXX 号”。
3.  **深色墨韵主题 (Dark Ink Theme)**: 
    *   主色调从“宣纸白” (`#fffdf5`) 切换为“玄色/墨色” (`#1a1a1a`)。
    *   文字颜色从“深褐” (`#3e2723`) 切换为“牙白/金” (`#d7ccc8`/`#d4af37`)。
    *   增加半透明的黑色磨砂玻璃质感。

## 1. 界面元素优化

### A. 标题与页脚
*   `Archives Total` -> `奏折总计`
*   `Memorials Archive` -> `文书省存档`
*   `No records` -> `暂无奏折记录`

### B. 编号重构
*   `#6810` -> `第 陆捌壹零 号` (或简单的 `第 6810 号`)

### C. 状态标签本地化
*   `completed` -> `【准奏】`
*   `rejected` -> `【驳回】`
*   `active` -> `【谕旨执行中】`
*   `planning` -> `【内阁拟旨中】`
*   `paused` -> `【暂缓执行】`
*   `cancelled` -> `【已撤回】`

## 2. 色彩与纹理调整

*   **面板背景**: `bg-[#1a0f0f]/90` + `backdrop-blur-lg`。
*   **列表项背景**: 从 `bg-white` 改为 `bg-black/40`，边框改为 `#d4af37/20`。
*   **进度条**: 保持金色，但底色改为深色，增加发光效果。
*   **内部区域**: `bg-[#2c1e1a]/50` (深木色) 替代原本的浅灰色背景。

## 3. 实施步骤

1.  **修改 `MemorialsPanel.tsx`**:
    *   更新 `renderStatusTag` 函数，实现完整的状态翻译。
    *   更新 ID 渲染逻辑。
    *   重写 Tailwind 类名，应用深色主题。

2.  **样式微调**:
    *   确保在 `variant="embedded"` 模式下的间距与新的 `PanelFrame` 完美契合。
    *   调整滚动条样式为深色。

## 4. 预期效果

*   界面风格与整个“指挥舱”完美融合。
*   用户能够直观理解每一个奏折的状态。
*   整体视觉体验更加沉浸、严肃。
